      • Explicit cert and WAF ownership

      HA / DR

      Platform HA

      • Redundant management-domain infrastructure
      • Multiple NSX Managers
      • Redundant Edge nodes
      • Resilient shared platform services

      LB control-plane HA

      • 3-node Avi Controller cluster
      • Dedicated control-plane health monitoring
      • Admin access and break-glass model
      • Backup and restore tested

      LB data-plane HA

      • Multiple Service Engines
      • Service Engine Groups aligned to failure domains
      • Frontend VIP redundancy
      • Deterministic routing and return path

      HA / DR validation matrix

      ScenarioWhat to testExpected result
      Controller lossController cluster resiliency, admin continuity, policy retentionManagement recovers without uncontrolled service impact
      Service Engine lossData-plane failover and health-monitor responseVIP traffic fails over within documented behavior
      Edge lossTier-0/Tier-1, north-south routing, NAT dependencyRouting and stateful services remain within HA design
      Upstream route failureBGP resiliency and alternate path behaviorNorth-south traffic converges as designed
      Certificate failureExpired or rotated certificate handlingDocumented alerting and rotation process works
